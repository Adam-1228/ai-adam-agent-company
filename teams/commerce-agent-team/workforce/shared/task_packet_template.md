# Task Packet Template

아래 양식으로 에이전트에게 업무를 배정합니다.

```text
TASK ID:
OWNER:
STATUS: TODO
PRIORITY: P1/P2/P3
REQUEST:
INPUTS:
EXPECTED OUTPUT:
DEADLINE:
CONSTRAINTS:
NEXT HANDOFF:
```

## 예시

```text
TASK ID: TASK-20260521-001
OWNER: 01_market_scout
STATUS: TODO
PRIORITY: P1
REQUEST: 쿠팡 사무용품 카테고리에서 소형/저위험 상품 후보 20개 조사
INPUTS: 목표 판매가 7,000~20,000원, 인증 리스크 낮은 상품 우선
EXPECTED OUTPUT: 후보 CSV 20개, 출처 URL, 리스크 메모
DEADLINE: 오늘
CONSTRAINTS: 전기/식품/화장품/캐릭터 상품 제외
NEXT HANDOFF: 02_margin_analyst
```
