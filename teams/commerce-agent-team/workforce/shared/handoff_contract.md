# Handoff Contract

에이전트 간 인수인계는 이 계약을 따릅니다.

## Scout -> Analyst

필수 전달:

- 상품명
- 카테고리
- 후보 출처
- 목표 판매 플랫폼
- 예상 소싱가
- 예상 판매가
- 경쟁 수
- 리뷰 수/평점
- 고객 불만 키워드
- 제외해야 할 리스크 메모

## Analyst -> Risk Guardian

필수 전달:

- 예상 마진
- 마진율
- 수요 점수
- 경쟁 점수
- 총점
- 숫자로는 좋아 보이지만 확인이 필요한 항목

## Risk Guardian -> Listing Builder

필수 전달:

- 진행 가능 여부
- 하드 스톱 여부
- 금지 표현
- 필요한 증빙
- 상세페이지에 반드시 넣어야 할 구매 전 확인사항

## Listing Builder -> Ops Manager

필수 전달:

- 상품명 후보
- 상세페이지 섹션
- FAQ
- 검색 키워드
- 검수 필요 문장
- 등록 전 체크리스트

## Ops Manager -> Adam

필수 전달:

- 진행/검토/보류/폐기 목록
- 가장 중요한 리스크
- 다음 액션
- 사람이 승인해야 할 항목
